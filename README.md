# Palopi Estudio Landing Page

Official landing page of Palopi Estudio - Digital Studio specializing in Website Development, UI/UX Design, and Statistical Data Analysis.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Vite (Dev Server & Bundler)

## Development
To run locally:
```bash
npm install
npm run dev
```

To build production files:
```bash
npm run build
```
